# TWK_KBE58_SSI
  This Code reads the SSI absolute encoder inside a Rohde &amp; Schwarz RD130   rotor. The encoder is assumed to be a TWK KBE 58 - K 4096 G K E06 with Gray   code, 12 useful position bits and 13 SSI clocks. The interface between the ESP32 and the encoder uses one ADM3490ARZ full   duplex RS422 transceiver.
